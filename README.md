# quickmeet
A video chat/meeting webapp using WebRTC and WebSockets (socket.io).
<br><br>

## How to run the application

-   Install Packages 
```base 
npm install
```

- Run the application
```base
npm start
```

## Current Features?

1. Video Calls
2. Chatbox
3. Collaborative Whiteboard
4. Screensharing

## Upcoming Features?

1. User Login/Signup
3. List of attendees
